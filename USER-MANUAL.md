# User Manual - DocVerifier

## Quick Start

**Generate Hash:**
1. Visit site
2. Upload document  
3. Copy/share 64-char SHA-256 hash

**Verify:**
1. Upload document → get hash A
2. Paste expected hash B  
3. A===B → Valid document

## FAQ
**Free?** Yes, always
**Private?** Yes, client-only
**Works offline?** Yes after first load
**File types?** PDF/DOC/JPG/PNG
